# CLAUDE.md

## Language

All files tracked by git must be written in English.

## Git

Keep history clean and reviewable before pushing. Squash fixup commits
(spacing tweaks, typo corrections, incremental adjustments) into the
feature or fix commit they belong to using amend or squash. Each commit
must represent a coherent, independently reviewable unit of change.
Commit messages describe the feature-level change as if written fresh —
no mention of the amend or squash process.

Do not squash unrelated changes together. Do not squash when doing so
would require conflict resolution.
